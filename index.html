Hardware Maintenance

Keeping older machines useful through diagnosis, repair and upgrades

I have repaired, rebuilt and upgraded computers for many years because I genuinely enjoy understanding how machines fail and how to bring them back to life. My work includes older Apple hardware, Linux installs, storage upgrades, internal cleaning, diagnostics and board-level repair experiments.

  • 2012 27-inch iMac GPU replacement using a custom-firmware Nvidia card with Metal support.
  • MacBook Pro GPU fault recovery by disabling defective discrete graphics and using the stable graphics path.
  • Micro-soldering work including a custom CMIzapper Tiresias board fitted to a logic board.
  • Retrofit workflows to run newer macOS versions on older, still-capable Macs.
